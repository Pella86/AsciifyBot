## Technologies
* Python 3.10
* Requests 2.27.1

## Features
* The bot is running on http requests using the [Telegram bot API](https://core.telegram.org/bots/api)
* The bot has no external library a part from [Requests](https://requests.readthedocs.io/en/latest/)
